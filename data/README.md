# Dataset

## Garbage Classification Dataset

Dataset yang digunakan untuk training model klasifikasi sampah.

**Sumber:** [Kaggle - Garbage Classification](https://www.kaggle.com/datasets/asdasdasasdas/garbage-classification)

**Lisensi:** CC0 Public Domain

### Struktur Dataset
```
TRAIN/
├── O/   → 12.565 gambar (Organik)
└── R/   → 9.999 gambar (Anorganik/Recyclable)
```

**Total:** 22.564 gambar

### Cara Download
1. Buka link Kaggle di atas
2. Download dataset
3. Ekstrak ke Google Drive: `MyDrive/DATASET/TRAIN/`

> Dataset tidak di-upload ke GitHub karena ukurannya terlalu besar (~2 GB).
